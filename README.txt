Blaziken TCG Checklist iPhone App

This folder is an installable iPhone web app, also called a PWA. It includes the 32 unique English Blaziken card numbers plus tracked English print and finish variants.

Files:
- index.html: the app
- manifest.json: app install metadata
- sw.js: offline cache
- icon.svg: app icon

Card details included:
- Card name
- Set/bundle name
- Collector number
- Rarity
- Holo/non-holo finish label

Current checklist size:
- 32 unique English Blaziken card numbers
- 55 English checklist entries when print and finish variants are counted

Variant examples included:
- EX Emerald 1/106 Holo
- EX Emerald 1/106 Reverse Holo
- EX Emerald 1/106 Non-Holo from Wildfire Theme Deck
- EX Emerald 1/106 Non-Holo from Typhlosion ex Epic Collection Deck

To preview on this computer:
Open index.html in a browser.

To install on iPhone:
1. Put this folder on a static HTTPS host such as GitHub Pages, Netlify, Vercel, Cloudflare Pages, or your own website.
2. Open the hosted URL in Safari on the iPhone.
3. Tap Share.
4. Tap Add to Home Screen.
5. Tap Add.

Note:
iOS does not install arbitrary .ipa files without Apple signing. Publishing a native App Store/TestFlight app requires a Mac with Xcode and an Apple Developer account.
